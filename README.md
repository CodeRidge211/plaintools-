# plaintools-

**The Plain Tools** — Free, No Signup Utility Hub  
*By Sovereign Ridge Partners LLC*

## Stack
- **Framework:** Next.js 14 (App Router)
- **Styling:** styled-jsx + CSS Variables
- **PDF Processing:** pdf-lib (client-side)
- **Design:** Deep Slate & Cyan dark theme, glassmorphism, mobile-first

## Features
- 📄 PDF Compress & Merge (in-browser, zero upload)
- 🕵️ Anonymous Tip Form (no IP logging)
- 📋 Invoice & NDA Templates (live preview + PDF download)
- 🔍 Smart Search Router (natural language → tool)
- ✨ Plain English Mode (toggle helper text for non-tech users)

## Run Locally
```bash
npm install
npm run dev
```

## Philosophy
> No signup. No file storage. All processing in-browser. Plain English everywhere.
